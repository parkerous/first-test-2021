# Roblox Volleyball Self-Study — BVL & CVR

Self-study notes on two realistic Roblox volleyball games, compiled from online
guides and gameplay tutorials. Focus: **spike, set, block, receive** (plus serve
and dive). Binsu has its own file: [`binsu.md`](./binsu.md).

## 1. BVL — Beyond Volleyball League

One of the most popular realistic Roblox volleyball games. Control scheme is the
"classic" Roblox volleyball layout that CVR and others also borrow from.

### Controls
| Action | Key |
|---|---|
| Spawn a ball | `G` |
| Basic serve | `V` |
| Sprint | `T` |
| Adjust power | `Q` / `E` |
| Set forward / backward | `F` / `R` |
| Receive (on ground) | Left Click |
| Dive | `3` |
| Toggle spike / block (while jumping) | `1` |
| Spike (jumping, spike mode) | Left Click |
| Curve on serve | `L` / click |

### The four core skills

**Spike**
- Toggle to spike mode with `1`, jump, then click *in the air* to hit.
- Timing is everything: swing to meet the ball at the **highest point** of your
  jump. Hitting too early or late kills power and accuracy.
- Use `Q`/`E` to set power *before* the swing so the ball clears the net but
  stays in bounds.

**Set**
- `F` = set forward, `R` = set backward.
- The set feeds the spiker. Aim for a **consistent, high, slightly-off-the-net**
  ball so your spiker can meet it at peak. Accuracy > flashiness.
- Good rule: every good spike starts with a good set.

**Block**
- Same `1` toggle, but land in **block mode** when you jump at the net.
- Jump timed to the opponent's peak contact point — slightly *after* the spiker
  jumps, so your hands are up when they swing.
- A read block (watch the setter, then the hitter) beats a guess block.

**Receive (bump / dig)**
- Click while on the ground to receive.
- Get **under** the ball, feet set, and tap it up toward your setter rather than
  smashing it back.
- Consistent, controlled timing beats spamming clicks.

### Practice priorities
1. Serve → receive consistently (get the ball to the setter).
2. Set to a target spot every time.
3. Spike timing at peak jump.
4. Only then add blocking reads and dives (`3`).

## 2. CVR — Conference Volleyball Roblox (a.k.a. "CVR: Retro")

A more technical/realistic experience with skills, abilities and builds layered
on top of the same base control scheme as BVL.

### Controls
| Action | Key |
|---|---|
| Spawn a ball | `G` |
| Basic serve | `V` |
| Sprint | `T` |
| Adjust power | `Q` / `E` |
| Set forward / backward | `F` / `R` |
| Receive (on ground) | Left Click |
| Dive | `3` |
| Toggle spike / block (while jumping) | `1` |
| Spike (jumping) | Left Click |
| Toggle trait ability | `Y` |
| Setter ping | `H` |

### What's different vs BVL
- **Traits / abilities** (`Y`): characters/builds have skills you toggle. Learn
  which trait fits your role (setter, spiker, libero) before ranking up.
- **Setter ping** (`H`): communicate where you want the ball — teamwork matters
  more here than in casual games.
- **Jump serve**: a higher-skill serve (jump + serve) for extra power/spin; worth
  practicing once basic serve is reliable.
- Same four fundamentals (spike/set/block/receive) — the extra depth is in
  abilities, positioning and team coordination.

### CVR-specific tips
- It's "technical": expect a steeper learning curve. Master the base bump-set-spike
  loop first, *then* layer abilities.
- Use setter ping so your team knows the play — reduces bad sets.
- There's a Trello board with builds, abilities and mechanics — good reference as
  you progress.

## Shared fundamentals (both games)

- **Watch the ball from the serve** — track it the whole rally.
- **Timing > spamming** — one well-timed click beats five mistimed ones.
- **Peak-of-jump contact** for both spikes and blocks.
- **Bump-set-spike** is the rhythm of every good rally: receive to the setter,
  set to the hitter, spike over.
- Adjust **power** (`Q`/`E`) deliberately — most beginner errors are hitting long
  (out) or into the net.
